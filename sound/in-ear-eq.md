# Setting In-Ear EQ for a Specific Output

1. On streamdeck press "Sound" button to get to sound pages
1. Press "In-Ear EQ Set" button
1. Press button for location to be setup
  - This will automatically switch to another page, if it does not happen right away press the location again
1. Press the in-ear type on the top to set the eq for that location and type
  - You should see the location on the bottom left, if it is incorrect press cancel
  - Press reset to reset the EQ for that location
