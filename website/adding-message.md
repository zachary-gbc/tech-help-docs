# Adding Message to Website
View prior messages to see format and get examples

1. Open Squarespace
1. Navigate to Semon blog (in Sermons Folder)
1. Click + to add a new sermon
1. Add message title (usually copied from sermon outline)
1. Press blue + below title and choose video
  - If making before Sunday leave video blank and will be added once stream has been updated on YouTube
1. Press + below video and choose button
  - Choose “file” and upload sermon outline pdf and set button text to Sermon Outline
1. Press + below button and choose button
  - Choose “file” and upload small group questions pdf and set button text to Small Group Questions
1. Drag the second button from bottom up to be next to first button
1. Click Save on the top left to save the message
1. On the left, click the three dots by the message just added and click settings
1. Click Content
  - Change the author to the speaker for the day
1. Click Options
  - Change the Status to scheduled for the Sunday of the service at 12:00pm
  - Under Tags, add the chapter of the Bible being covered if message about a specific passage, if topical message skip this
  - Under Categories add the message series and if special event like baptism or child dedication
1. Click Save to save settings for message

Special Cases:
-	If baptism Sunday need to add another button to link to the testimonies
-	If more than one service is streamed (such as for baptisms or child dedications) add more videos, one for each service
