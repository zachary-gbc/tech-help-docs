# Nova Star (LED Wall) Settings
Sometimes the settings for the Nova Star get changed and need to be reset. Process and values are below:

1. Press Menu
1. Brightness should be set to 30%
1. Go to Ouptut Settings
1. Scaling should be set to Enabled
1. Auto Fit should be set to Disable
1. Go to Custom Scaling
1. Go to Input Capture
1. Width should be set to 1920
1. Height should be set to 1080
1. Escape and go to Output Window
1. Width should be 1144
1. Height should be 624
1. Press Escape multiple times to exit the menu (settings are automatically saved)
